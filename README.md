# InnerWhispers FD-1

Frontend Authentication and Profile Module for the InnerWhispers Wellness Companion.

## Completed Features

### Authentication

- Login Page
- Register Page
- Password Strength Indicator
- Forgot Password Page
- OTP Verification Page
- Reset Password Page

### Profile

- Avatar Upload Placeholder
- Dynamic Initial-Based Avatar
- Display Name Section
- Bio Section
- Settings Link
- Remove Profile Photo Option

## Design System

- Primary Color: #E8724A
- Background: #F2F0EE
- Cards: #FFFFFF
- Primary Text: #2C2C2C
- Muted Text: #9A9080

## Tech Stack

- HTML5
- Tailwind CSS
- JavaScript
